# Weight-Label

중량계에 값을 가져와서 프린트로 인쇄하기

### Install

```
git clone https://github.com/tdat97/Weight-Label.git
cd Vision-Reader
pip install -r requirments.txt
```

---

### Run

```
python run.py
```
